# Landing Page - Aplicación y Manipulación Fotográfica UAN

![Banner UAN](images/screenshot.jpg)

## Descripción

Este proyecto es una landing page moderna para el curso de "Aplicación y Manipulación Fotográfica" de la Universidad Antonio Nariño. Diseñada para estudiantes de Diseño Gráfico, esta página muestra información detallada sobre el curso, sus objetivos, metodología y contenidos.

## Características

- **Diseño Responsivo**: Adaptable a diferentes dispositivos (móviles, tablets y desktops)
- **Menú Lateral**: Con navegación fácil y accesible
- **Efectos Visuales**: Animaciones y transiciones suaves para elementos interactivos
- **Contenido Estructurado**: Secciones bien definidas para cada aspecto del curso
- **Paleta de Colores**: Utiliza la identidad visual de la UAN con toques modernos
- **Optimización SEO**: Estructura HTML semántica para mejor indexación
- **Accesibilidad**: Elementos accesibles mediante teclado y lectores de pantalla

## Tecnologías Utilizadas

- HTML5
- CSS3 (Flexbox y Grid)
- JavaScript (ES6+)
- Font Awesome (para iconos)
- Diseño responsive sin frameworks externos

## Instalación y Uso

1. Clona este repositorio:
   ```
   git clone https://github.com/tu-usuario/uan-foto-landing.git
   ```

2. Abre el archivo `index.html` en tu navegador para ver la página.

3. Para hacer modificaciones:
   - Edita `index.html` para cambiar la estructura o el contenido
   - Modifica `css/styles.css` para ajustar los estilos
   - Actualiza `js/main.js` para cambiar las funcionalidades

## Vista Previa

La página incluye:

- **Header con Imagen de Fondo**: Presenta el título del curso con una cita inspiradora
- **Sección de Bienvenida**: Introducción al curso con imágenes comparativas
- **Aprendizaje**: Listado de habilidades a desarrollar
- **Unidades Temáticas**: Detalle de las tres unidades principales con ejemplos visuales
- **Metodología**: Enfoque pedagógico con iconos animados
- **Requisitos y Evaluación**: Información práctica para los estudiantes
- **CTA**: Llamado a la acción para inscripción
- **Información**: Datos del docente y del curso
- **Footer**: Información institucional y enlaces de contacto

## Personalización

Este proyecto puede personalizarse fácilmente para otros cursos o instituciones:

1. Cambiar la paleta de colores en `css/styles.css` modificando las variables CSS:
   ```css
   :root {
       --color-primary: #1DF5D1;    /* Turquesa */
       --color-secondary: #003366;  /* Azul UAN */
       --color-tertiary: #FF4C76;   /* Rosa complementario */
   }
   ```

2. Reemplazar las imágenes en la carpeta `/images` manteniendo los mismos nombres de archivo.

3. Modificar el contenido en `index.html` según las necesidades del curso.
